# Data-analytics-project-
A data analysis-based customer churn prediction system that analyzes customer behavior and predicts potential churn to help businesses improve retention strategies.
📌 Overview
A data-driven system to predict whether a telecom customer is likely to churn, built using Python with a Decision Tree classifier. The system covers the full ML pipeline — from data preprocessing to a Flask web interface for real-time predictions.
🚀 Features
Automated data preprocessing (encoding, scaling, SMOTE balancing)
Exploratory Data Analysis with visualizations (pie charts, box plots, heatmaps)
Decision Tree classifier with accuracy, precision, recall, and F1 evaluation
Feature importance analysis
Flask web app for instant churn risk prediction
Key Insights:
Short-tenure customers churn more
Month-to-month contracts have the highest churn rate
Higher monthly charges correlate with increased churn risk
